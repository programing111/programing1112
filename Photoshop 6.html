Mengenal Fungsi Tools Pada Photoshop


1. Move Tool (V): Memindahkan objek atau layer. Digunakan untuk menggeser objek di dalam dokumen Photoshop.

2. Marquee Tool (M): Membuat seleksi berbentuk persegi atau lingkaran. Terdiri dari:
- Rectangular Marquee Tool: Membuat seleksi berbentuk persegi atau persegi panjang.
- Elliptical Marquee Tool: Membuat seleksi berbentuk lingkaran atau oval.

3. Lasso Tool (L): Membuat seleksi bebas secara manual. Ada beberapa jenis:
- Lasso Tool: Membuat seleksi secara bebas.
- Polygonal Lasso Tool: Membuat seleksi dengan garis lurus.
- Magnetic Lasso Tool: Membuat seleksi otomatis mengikuti tepi objek.

4. Magic Wand Tool (W): Menyeleksi area dengan warna serupa hanya dengan satu klik.

5. Crop Tool (C): Memotong bagian gambar sesuai area seleksi yang dipilih.

6. Eyedropper Tool (I): Mengambil warna dari area tertentu dalam gambar dan menggunakannya sebagai warna foreground atau background.

7. Brush Tool (B): Menggambar atau melukis dengan sapuan kuas yang dapat diatur ukuran dan teksturnya.

8. Clone Stamp Tool (S): Menyalin sebagian gambar dan menggandakannya di bagian lain. Sangat berguna untuk memperbaiki gambar dengan menghilangkan objek yang tidak diinginkan.

9. Healing Brush Tool (J): Memperbaiki area gambar yang rusak atau cacat dengan mencocokkan tekstur di sekitarnya. Ada dua varian:
- Spot Healing Brush Tool: Memperbaiki area kecil secara otomatis.
- Patch Tool: Memperbaiki area besar dengan memilih sumber perbaikan dari bagian lain gambar.

10. Eraser Tool (E): Menghapus bagian gambar atau layer tertentu. Terdapat juga Background Eraser Tool yang menghapus latar belakang secara otomatis.

11. Gradient Tool (G): Membuat transisi warna secara bertahap antara dua atau lebih warna.

12. Paint Bucket Tool (G): Mengisi area dengan warna solid berdasarkan pilihan warna yang telah diatur.

13. Pen Tool (P): Membuat bentuk atau jalur vektor yang dapat digunakan untuk seleksi atau menggambar bentuk presisi.

14. Type Tool (T): Menambahkan teks ke dalam gambar. Terdiri dari:
- Horizontal Type Tool: Menambahkan teks secara horizontal.
- Vertical Type Tool: Menambahkan teks secara vertikal.

15. Path Selection Tool (A): Memilih dan memodifikasi jalur atau bentuk vektor.

16. Shape Tool (U): Membuat bentuk-bentuk dasar seperti persegi, lingkaran, garis, atau bentuk bebas lainnya.

17. Zoom Tool (Z): Memperbesar atau memperkecil tampilan gambar.

18. Hand Tool (H): Menggeser tampilan gambar ketika diperbesar (zoom in).

19. Blur Tool: Memburamkan area tertentu pada gambar untuk mengurangi ketajaman.

20. Dodge and Burn Tools:
- Dodge Tool: Mencerahkan area gambar.
- Burn Tool: Menggelapkan area gambar.

21. Smudge Tool: Mengaburkan piksel gambar dengan cara menyeretnya untuk memberikan efek campuran.

22. History Brush Tool: Mengembalikan bagian gambar ke kondisi sebelumnya berdasarkan riwayat pengeditan.

23. Sponge Tool: Menambah atau mengurangi saturasi warna di area tertentu pada gambar.

24. Ruler Tool: Mengukur jarak antara dua titik atau membuat garis panduan untuk presisi pengeditan.
