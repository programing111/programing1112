Cara Membuat Jadwal Pelajaran dengan HTML


Membuat jadwal pelajaran dengan HTML bisa dilakukan dengan mudah menggunakan tabel. Tabel di HTML didefinisikan dengan tag <table>, dan terdiri dari baris <tr>, kolom <td>, dan header tabel <th>. Berikut adalah contoh sederhana cara membuat jadwal pelajaran:

Contoh kode HTML:

<table border="1">
  <tr>
    <th>Hari</th>
    <th>Mata Pelajaran</th>
    <th>Waktu</th>
  </tr>
  <tr>
    <td>Senin</td>
    <td>Matematika</td>
    <td>08:00 - 09:30</td>
  </tr>
  <tr>
    <td>Selasa</td>
    <td>Bahasa Indonesia</td>
    <td>08:00 - 09:30</td>
  </tr>
</table>

Dalam contoh di atas, tag <table> digunakan untuk membuat tabel, sementara <tr> digunakan untuk membuat baris, dan <td> untuk menampilkan data di setiap kolom.
