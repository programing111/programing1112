Mengenal HTML 5


HTML (Hypertext Markup Language) adalah bahasa markup yang digunakan untuk membuat dan mengatur halaman web. HTML menyediakan struktur dasar sebuah halaman web dengan elemen-elemen yang ditulis dalam bentuk "tag". Setiap tag HTML memiliki fungsi spesifik yang membantu dalam menampilkan konten seperti teks, gambar, video, link, dan sebagainya di browser.

Struktur Dasar HTML: <!DOCTYPE html>
<html lang="en">
<head>
  <meta charset="UTF-8">
  <meta name="viewport" content="width=device-width, initial-scale=1.0">
  <title>Title of the page</title>
</head>
<body>
  <h1>Hello, World!</h1>
  <p>Ini adalah halaman HTML pertama saya.</p>
</body>
</html>

- <!DOCTYPE html>: Deklarasi yang memberitahu browser bahwa dokumen ini menggunakan HTML5.
Contoh:

<!DOCTYPE html>

- <html>: Tag pembuka yang menyatakan awal dari dokumen HTML.
Contoh:

<html lang="en">
...konten halaman...
</html>

- <head>: Bagian dari halaman HTML yang berisi meta informasi, seperti judul halaman (<title>) dan referensi ke stylesheet atau skrip.
Contoh:

<head>
  <meta charset="UTF-8">
  <title>Judul Halaman</title>
</head>

- <meta charset="UTF-8">: Menentukan encoding karakter yang digunakan (UTF-8 mencakup hampir semua karakter dan simbol).
Contoh:

<meta charset="UTF-8">

- <title>: Menentukan judul halaman yang akan muncul di tab browser.
Contoh:

<title>Judul Halaman</title>

- <body>: Berisi konten utama halaman web, seperti teks, gambar, dan elemen interaktif lainnya.
Contoh:

<body>
  <h1>Ini adalah judul</h1>
  <p>Ini adalah paragraf.</p>
</body>

Tag-Tag HTML Penting:
1. Heading Tags (h1 - h6): Heading digunakan untuk membuat judul atau subjudul pada halaman web.
Contoh:

<h1>Ini adalah Heading 1</h1>
<h2>Ini adalah Heading 2</h2>

2. <p> (paragraph): Digunakan untuk menulis paragraf teks.
Contoh:

<p>Ini adalah sebuah paragraf dalam HTML.</p>

3. <a> (anchor): Digunakan untuk membuat hyperlink yang mengarahkan ke halaman lain atau URL.
Contoh:

<a href="https://www.example.com">Kunjungi Example</a>

4. <img> (image): Digunakan untuk menampilkan gambar di halaman web.
Contoh:

<img src="image.jpg" alt="Deskripsi Gambar">

5. <video> (video): Digunakan untuk menyematkan video ke dalam halaman web.
Contoh:

<video src="video.mp4" controls></video>

6. <ul> (unordered list): Digunakan untuk membuat daftar dengan bullet points.
Contoh:

<ul>
  <li>Item 1</li>
  <li>Item 2</li>
  <li>Item 3</li>
</ul>

7. <ol> (ordered list): Digunakan untuk membuat daftar dengan nomor urut.
Contoh:

<ol>
  <li>Item 1</li>
  <li>Item 2</li>
  <li>Item 3</li>
</ol>

8. <li> (list item): Digunakan untuk item di dalam daftar (<ul> atau <ol>).
Contoh:

<li>Item Daftar</li>

9. <table> (table): Digunakan untuk membuat tabel.
Contoh:

<table>
  <tr>
    <th>Header 1</th>
    <th>Header 2</th>
  </tr>
  <tr>
    <td>Data 1</td>
    <td>Data 2</td>
  </tr>
</table>

10. <tr> (table row): Digunakan untuk mendefinisikan baris dalam tabel.
Contoh:

<tr>
  <td>Data</td>
</tr>

11. <td> (table data): Digunakan untuk mendefinisikan sel data dalam tabel.
Contoh:

<td>Data</td>

12. <th> (table header): Digunakan untuk mendefinisikan sel header dalam tabel.
Contoh:

<th>Header</th>

13. <link> (link): Digunakan untuk menghubungkan dokumen dengan stylesheet eksternal atau sumber daya lain.
Contoh:

<link rel="stylesheet" href="styles.css">
