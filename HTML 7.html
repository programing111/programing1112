HTML, singkatan dari Hypertext Markup Language, adalah bahasa markup yang digunakan untuk membuat dan menyusun halaman web. HTML menyusun dokumen dengan menggunakan berbagai tag yang memberikan struktur dan makna pada konten web. Setiap halaman web dimulai dengan deklarasi <!DOCTYPE html> yang memberi tahu browser bahwa dokumen ini mengikuti standar HTML5.

Struktur dasar HTML terdiri dari elemen-elemen berikut:

- **Tag ``**: Ini adalah elemen root dari dokumen HTML. Semua elemen lain berada di dalam tag ini. 

<html>
    <head>
        <title>Judul Halaman</title>
    </head>
    <body>
        <h1>Ini Judul</h1>
        <p>Ini paragraf</p>
    </body>
</html>


- **Tag ``**: Bagian ini berisi metadata tentang dokumen, termasuk judul halaman dan link ke stylesheet atau script eksternal. 
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Judul Halaman</title>
    <link rel="stylesheet" href="style.css">
</head>


- **Tag ``**: Berisi konten utama yang ditampilkan di halaman web, seperti teks, gambar, dan video. 
<body>
    <h1>Judul Utama</h1>
    <p>Teks paragraf</p>
    <img src="gambar.jpg" alt="Deskripsi gambar">
    <video controls>
        <source src="video.mp4" type="video/mp4">
        Your browser does not support the video tag.
    </video>
</body>
